# Runtime dependencies for the deployed Azure Functions app.
#
# Flex Consumption remote-builds the app from this file (ADR-0004). This is the
# transitive third-party closure of the entrypoints in function_app.py:
#   - health          (stdlib)
#   - scheduled_ingestion (#20)  -> shared, dataverse, openf1  -> httpx, azure-identity
#   - predict (#57)             -> ml.serving                 -> scikit-learn, pandas, joblib
#   - config/secrets            -> shared.config, credentials  -> pydantic(-settings), azure-keyvault
#
# NOTE: importing ml.serving runs ml/__init__.py, which eagerly imports the whole
# ML pillar — ml.clustering pulls in *matplotlib*, and ml.features imports
# fastf1_analytics -> *fastf1*. So the deployed app genuinely needs both, even
# though only /predict uses the ML path. (Omitting them made the host fail to
# import function_app.py at index time -> zero functions registered -> 404.)
#
# The project's own packages (ai, ml, rag, agents, shared, dataverse, openf1,
# paddock, azure_functions) ship as deployment content — src/ is the app root,
# so they import directly.
